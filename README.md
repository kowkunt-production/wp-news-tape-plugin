# wp-news-tape-plugin

The WP News Tape Plugin is designed to display dynamic, ticker-style
content feeds for breaking news and curated information. 

Content is sourced from multiple providers based on
the selected location, category, and keywords.


- [ ] [Display ads and news in a news tape.](https://github.com/kowkunt-production/wp-news-tape-plugin/blob/main/README.md)

- [ ] News from different sources for different sites based on the users' location, selected category and keywords.
